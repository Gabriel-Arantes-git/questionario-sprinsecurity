# questionario-sprinsecurity


1- Spring Security é uma biblioteca com funcionalidades de controle de requisições, gerenciando acessos, autenticações, maneira de login e dados para gestão de acesso às requisições do backend

2- para desabilitar a criacao de um token de protecao csfr.

3- Significa que a aplicação não deve guardar informações de login do usuário/requisitor, qualquer informação de autenticação deve ser enviada junto da requisição

4- Permite o acesso à todos os endpoints dentro do backend/servico.

5 - definine os criterios das requisicoes, filtrando e gerenciando acesso/restricoes para acessar um endpoint

6 - define que toda requisição deve ser autenticada dentro do backend

7 - que não armazena nenhuma informacao sobre os dados de autenticacao do usuario, necessitando que qualquer informacao seja enviada junto da requisição

8 - determina que cada seção deve ter como politica a execução seguindo os conceitos de stateless, nao armazenando os dados do usuário
